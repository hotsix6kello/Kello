# K-TRIP: Korea Travel Operating System

> "Kakao보다 먼저 켜는 외국인 전용 한국 생활/여행 슈퍼앱"

## 1. Project Vision
**K-TRIP**은 단순한 여행 정보 앱이 아닙니다.  
외국인이 한국에 도착하는 순간부터 출국까지 이동, 일정, 예약, 결제, 커뮤니티를 **AI가 실시간으로 운영(Operating)**해주는 OS입니다.

- **Status:** v3.0 (Integrated Final)
- **Key Feature:** Smart Trip Planner, Carecation, Transport Navigator

## 2. Tech Stack
- **Frontend:** Next.js (App Directory)
- **Styling:** Vanilla CSS / CSS Variables (Premium Glassmorphism)
- **Backend:** Supabase
- **AI:** OpenAI Integration

## 3. How to Run
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)
